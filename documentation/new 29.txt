1.) The veriations of getting a random number:
    a.) RANDNUM[ LEN-4 ] : returns a random number 4 digits in length.
    b.) RANDNUM[] : returns a random number of 0 through 9.
    c.) RANDNUM[ PREC-4-2 ] : returns a random number with the precision of (4.2).
    
2.) The veriation of getting a random letter:
    a.) RANDLET[ A-M ] : returns any upper case letter from A to M.
    b.) RANDLET[ a-m ] : returns any lower case letter from a to m.
    c.) RANDLET[ A-M LEN-4 ] : returns random upper case letters from A to M four characters in length.
    d.) RANDLET[ a-m LEN-4 ] : returns random lower case letters from a to m four characters in length.
    c.) RANDLET[ VOL ] : returns a random voul.
    d.) RANDLET[ CON ] : returns a random constinent.
